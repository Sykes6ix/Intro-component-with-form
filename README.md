# Intro Component with Sign-Up Form

This is my solution to the **Frontend Mentor Intro Component with Sign-Up Form challenge**. The goal of this project was to build a responsive landing section with a sign-up form using clean HTML and CSS.

## 📌 Overview

This project focuses on creating a modern and responsive intro section that includes:

* A marketing message
* A sign-up form
* Mobile and desktop layouts
* Basic form structure and styling

I completed the main layout and form structure. Error message validation will be added in a future update.



## 🚀 Live Demo

You can view the live project here:

https://sykes6ix.github.io/Intro-component-with-form/


## 🚀 Features

* Responsive design for mobile and desktop
* Clean and semantic HTML structure
* CSS Flexbox and/or Grid layout
* Styled form inputs
* Focus and hover states
* Modern UI design



## 🛠️ Built With

* Responsive design techniques
